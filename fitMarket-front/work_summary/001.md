# Work Summary
- 장바구니·상품에서 주문 시 새 체크아웃 페이지로 이동해 배송지 선택 후 결제하도록 라우팅과 CTA를 수정했어요.
- 배송지/주문 상태를 composable로 분리해 결제 진행·취소 흐름과 다중 주소 관리를 한 곳에서 제어하도록 만들었어요.
- 주문 완료 페이지는 주소 정보만 보여주고, 결제 완료 상태에서만 변경 가능하다는 문구와 버튼 상태를 명확히 안내하도록 재구성했어요.
- 결제 취소는 confirm 창으로 처리해 별도 페이지 이동 없이 상태를 변경하고, 취소/환불 알림을 사용자에게 즉시 전달하도록 했어요.
